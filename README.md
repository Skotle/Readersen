
## 자바 랭킹 크롤러

## 실행

* 첫 페이지 번호(PC기준), 마지막 페이지 번호 입력
* 페이지당 거의 50글
* main -메이저 m - 마이너 mini - 미니
* 스레드당 속도 최소 30 최대 50(8스레드부터 락)

## 가공

```bash
python ExcelPrinter.py
```
```bash
python GraphPrinter.py
```
* example.json, daily-data.txt에서 랭킹, 일일 데이터 커스텀 가능
